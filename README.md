# TE SU TO
## SE N SE I

### install server's packages
```
$ cd server
$ npm i
```

### run server (`localhost:3001`)
macOS:
```
$ PORT=3001 node bin/www
```
windows:
```
$ set PORT=3001 && node bin/www
```

### install application's packages
```
$ cd ..
$ npm i
```

### run application (`localhost:3000`)
```
$ npm start
```
